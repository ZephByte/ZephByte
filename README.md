## Hey, I'm ZephByte

I write code for whatever hobby has my attention that day.

Sometimes that means Minecraft mods. Sometimes it’s mobile apps, Mac apps, small IoT projects, or whatever else seems interesting enough to lose a weekend to.

My hobby seems to be hobbies.

### What I make

**Minecraft mods**  
I make server-side Fabric mods for Minecraft. One of my main projects is [ScaledDragonFight](https://modrinth.com/mod/scaled-dragon-fight), a mod that scales the Ender Dragon fight based on how many players are in The End.

Source code is available here: [ZephByte/ScaledDragonFight](https://github.com/ZephByte/ScaledDragonFight)

**Apps and tools**  
I like building small, practical software that I actually use, for the devices I actually use. Mobile, Mac, Windows, eReaders, and whatever else ends up on my desk.
